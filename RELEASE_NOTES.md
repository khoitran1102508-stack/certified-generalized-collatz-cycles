# Release notes: version 1.0.0

This stable artifact contains the final certificate and the exact source required to verify or regenerate it.
The certificate itself is unchanged from the independently audited certificate v2.

Public-facing filenames and documentation have been normalized for archival use. Development caches,
internal workflow reports, manuscript-editing tests, and obsolete experimental entry points are excluded.
The immutable certificate header retains its historical searcher identifier solely for provenance.
